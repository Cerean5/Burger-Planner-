# Burger Planner 🍔🌱

Burger Planner is a colorful vegan recipe planner PWA designed for burger lovers.

Features:
• Vegan burger recipes
• Smoothie and salad ideas
• Meal planning tools
• Works offline
• Installable like a mobile app
• Colorful social media ready recipes

Live App:
https://burger-planner.replit.app

Install the app on your phone and start planning delicious plant-based meals.
